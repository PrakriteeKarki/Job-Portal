📌 Features
👤 User Authentication System
User Registration with Email Verification
Email activation link for account activation
Login / Logout system
Google OAuth Login (Continue with Google)
Inactive users cannot log in
👥 User Roles
Job Seeker
Employer

Each role has different dashboard features.

🧑‍💼 Job Seeker Features
Search jobs (keyword & location-based)
Apply for jobs
View “My Applications” page

🏢 Employer Features
Create job advertisements
Edit and delete job posts
View applicants for each job
Download candidate CVs
Job advertisements with full details
Pagination 
Search functionality for filtering jobs
📬 Email System
Automatic email sent after registration
Email verification required for account activation
⚡ API & Performance Features
Request throttling implemented to prevent abuse
🛠️ Tech Stack
Backend: Django, Django REST Framework
Database: PostgreSQL 
Frontend: HTML, CSS
Authentication: Django Auth + Google OAuth
Email Service: SMTP (Gmail)
Version Control: Git & GitHub
